

# Web3 钱包测试场景全景图

### 一、 核心功能测试

#### 1.1 钱包创建与助记词管理
- **助记词生成**
  - 测试12词和24词助记词的生成
  - 验证助记词是否符合BIP39标准
  - 测试生成速度和在弱网下的表现
  - 验证不同生成路径下的助记词唯一性
  - 测试生成过程中的异常处理（如中断、内存不足）
- **助记词备份验证**
  - 测试强制用户抄写助记词流程
  - 验证助记词乱序排列确认功能
  - 测试助记词输入错误时的具体提示
  - 验证助记词大小写、空格敏感性处理
  - 测试助记词备份的跳过和返回机制
- **助记词存储安全**
  - 测试系统截屏和录屏的阻止
  - 验证助记词在内存中的及时清理
  - 测试应用切换到后台时的模糊处理
  - 验证助记词是否被意外存储到本地文件

#### 1.2 钱包导入与恢复
- **助记词导入**
  - 测试完整正确助记词导入的成功率
  - 验证错误助记词（错词、少词、多词）的导入失败
  - 测试助记词顺序错误的识别能力
  - 验证不同语言助记词的兼容性
  - 测试从其他主流钱包导入的兼容性
- **私钥导入**
  - 测试标准私钥格式（0x开头64字符）导入
  - 验证私钥校验和检查机制
  - 测试错误私钥（长度不对、非法字符）的拒绝
  - 验证私钥导入后的地址生成正确性
- **Keystore文件导入**
  - 测试JSON文件格式解析
  - 验证正确密码解锁Keystore文件
  - 测试错误密码的解密失败
  - 验证损坏Keystore文件的识别

#### 1.3 资产管理功能
- **余额显示**
  - 测试主网币余额实时准确性
  - 验证ERC20代币余额和价格显示
  - 测试余额刷新机制（手动/自动）
  - 验证零余额、大数额的格式化显示
  - 测试汇率转换的准确性
- **资产列表**
  - 测试代币的自动发现和显示
  - 验证自定义代币的添加和删除
  - 测试代币图标的加载和缓存
  - 验证资产列表的排序和筛选
- **价格图表**
  - 测试不同时间维度（24H、7D、1M）的价格图表
  - 验证图表数据与行情源的一致性
  - 测试图表交互（缩放、拖动）的流畅性

#### 1.4 转账交易功能
- **收款功能**
  - 测试地址二维码生成和扫描
  - 验证地址复制粘贴功能
  - 测试不同币种收款地址的一致性
  - 验证ENS域名解析和显示
- **转账流程**
  - 测试地址输入校验（格式、有效性）
  - 验证余额不足的提前提示
  - 测试Gas费估算的准确性
  - 验证交易预览信息的完整性
- **交易确认**
  - 测试交易二次确认对话框
  - 验证交易哈希的生成和显示
  - 测试交易取消和加速功能
  - 验证交易失败的原因分析

#### 1.5 DApp浏览器与连接
- **DApp发现**
  - 测试推荐DApp列表加载
  - 验证DApp搜索功能
  - 测试DApp分类和排序
  - 验证DApp图标和描述的显示
- **钱包连接**
  - 测试WalletConnect协议连接
  - 验证MetaMask兼容模式连接
  - 测试连接请求的授权流程
  - 验证已连接DApp的管理
- **交易签名**
  - 测试交易签名请求的显示
  - 验证消息签名的内容和格式
  - 测试签名拒绝和超时处理
  - 验证签名历史记录

### 二、 安全测试

#### 2.1 本地数据安全
- **密钥存储安全**
  - 测试助记词/私钥的加密存储
  - 验证密钥在设备备份中的排除
  - 测试密钥在内存中的加密状态
  - 验证应用卸载时的数据清理
- **敏感信息保护**
  - 测试截屏屏蔽功能
  - 验证键盘缓存的禁用
  - 测试日志中的敏感信息过滤
  - 验证剪贴板自动清理

#### 2.2 交易安全
- **钓鱼防护**
  - 测试恶意地址识别和警告
  - 验证高风险交易的风险提示
  - 测试合约交互的风险评估
  - 验证授权限额的强制设置
- **Gas费安全**
  - 测试异常Gas费的警告
  - 验证Gas费上限的设置
  - 测试交易滑点保护
  - 验证前置运行防护

#### 2.3 身份认证安全
- **生物识别**
  - 测试指纹/面部识别的集成
  - 验证生物识别失败的回退机制
  - 测试生物识别设置的持久化
- **PIN码/密码**
  - 测试PIN码强度要求
  - 验证连续错误尝试的锁定
  - 测试密码修改流程的安全性

### 三、 性能测试

#### 3.1 响应性能
- **启动性能**
  - 测试冷启动时间（< 2秒）
  - 验证热启动时间（< 1秒）
  - 测试初始化加载优化
- **交易性能**
  - 测试交易广播速度
  - 验证余额查询响应时间
  - 测试Gas估算响应时间

#### 3.2 资源消耗
- **内存使用**
  - 测试长时间运行的内存增长
  - 验证多钱包同时加载的内存管理
  - 测试大额交易历史的内存优化
- **电池消耗**
  - 测试后台同步的耗电控制
  - 验证连续交易的电量消耗
  - 测试网络请求的耗电优化

#### 3.3 网络性能
- **弱网测试**
  - 测试2G/3G网络下的功能可用性
  - 验证网络中断的优雅处理
  - 测试网络恢复后的自动同步
- **节点切换**
  - 测试主节点故障时的自动切换
  - 验证节点响应时间的监控
  - 测试自定义节点的连接稳定性

## 首次上线重点测试场景

对于首次上线，建议重点关注以下**高优先级场景**：

1. **端到端钱包创建流程** - 从安装到成功创建第一个钱包
2. **核心转账功能** - 完成一笔完整的转账交易
3. **助记词备份与恢复** - 验证钱包可成功恢复
4. **DApp连接与交互** - 与主流DApp完成连接和交易
5. **安全防护机制** - 验证关键安全功能正常工作
6. **异常情况处理** - 网络中断、交易失败等场景
7. **新用户引导体验** - 确保新手能够顺利理解和使用

这份测试清单涵盖了Web3钱包从技术实现到用户体验的各个方面，可以作为测试计划和测试用例编写的重要参考。在实际项目中，可以根据具体功能范围和优先级进行适当裁剪。
